My Sample Project

Demonstration of building & deploying a static site using VS Code → GitHub → Webhook → Jenkins → Docker.

CI / CD • Docker • Jenkins

What this project shows

A one-page site that explains how I created and deployed this profile. The repository contains the index.html (this page) and a Dockerfile. Pushing to GitHub triggers Jenkins via a webhook which builds and deploys the image.

1

Edit in VS Code

Write or update index.html and Dockerfile locally. Commit changes with Git.

2

Push to GitHub

Push to your GitHub repo (example: github.com/Yugendrarepo/my-sample-website). GitHub stores the source and fires a webhook on push.

3

Webhook triggers Jenkins

GitHub sends a push event to Jenkins at /github-webhook/. Jenkins starts the pipeline defined by the Jenkinsfile or pipeline config.

4

Jenkins builds Docker image

Jenkins checks out the repo, builds the Docker image using the Dockerfile, stops the old container (if any), then runs the new container.

5

Website updated automatically

Open http://YOUR_SERVER_IP:8080 to see the latest version served by the container.

Files included (visible in your repo)

index.html — this page (static content & instructions)
Dockerfile — contains how to serve this page with Nginx
Jenkinsfile — (optional) pipeline script for Jenkins
Tip: If Jenkins cannot access your machine directly, use ngrok to tunnel GitHub webhooks to your local Jenkins URL.

Quick commands

Run these in your VS Code terminal (adjust the repo URL):

Git / push 
git init
git add .
git commit -m "Initial commit — index + dockerfile"
git remote add origin https://github.com/Yugendrarepo/my-sample-website.git
git branch -M main
git push -u origin main
Dockerfile (example) 
FROM nginx:alpine
RUN rm -rf /usr/share/nginx/html/*
COPY index.html /usr/share/nginx/html/index.html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
Jenkinsfile (example) 
pipeline {
  agent any
  stages {
    stage('Checkout') {
      steps { git branch: 'main', url: 'https://github.com/Yugendrarepo/my-sample-website.git' }
    }
    stage('Build Image') {
      steps { sh 'docker build -t yugendra/sample-website:latest .' }
    }
    stage('Deploy') {
      steps {
        sh '''
          docker stop sample-container || true
          docker rm sample-container || true
          docker run -d --name sample-container -p 8080:80 yugendra/sample-website:latest
        '''
      }
    }
  }
}
