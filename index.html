What this project shows

A one-page site that explains how I created and deployed this project. The repository contains the index.html (this page) and a Dockerfile. Pushing to GitHub triggers Jenkins via a webhook, which builds and deploys the image.

  1. Edit in VS Code Write or update index.html and Dockerfile locally. Commit changes with Git.

  2. Push to GitHub
    GitHub stores the code and fires a webhook event on push.

  3. Webhook triggers Jenkins
    Jenkins starts the pipeline using the Jenkinsfile or pipeline config.

  4. Jenkins builds Docker image
    Jenkins runs docker build to generate a new website image.

  5. Jenkins deploys container
    The old container is removed and the new one runs on port 8081.
Project Summary

This page documents how the project was created and deployed. It is intentionally simple and self-contained — ideal for CI/CD learning and interviews.
